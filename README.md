## Prerequisites

You need to have the following programs installed on your machine:
- [Node.js](https://nodejs.org/) (>= 8.11.3)
- [Yarn](https://yarnpkg.com/)

## Install

```sh
yarn
```

## Run

```sh
# Launch server with the React app at http://localhost:3000
yarn start:react
```
